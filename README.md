# 介绍
一个知识库问答系统，用户上传文档，然后存储在某个位置，当用户发起提问时，就会检索出文档中相似的片段然后将其作为上下文传给大模型，大模型就会结合用户提问和检索出的片段进行回答。

# 技术栈
nextjs、tailwind、react、typescript。
ai sdk，此sdk封装了一些服务，可以和大模型对话、还有rag的向量化和rerank等功能。

# 开发步骤
## 1.创建2个页面，包括 文档上传页，问答页。
文档上传后可保存在项目某个文件夹中（功能要实现）。问答页可以在提问时选择这个文件文档，然后提问（这里随便回答，ui上体现即可，不需要接入大模型sdk）

## 2.接入sdk，实现连接大模型然后进行简单的回答。

## 3.实现向量化方法当我提问时，返回相似度高的片段，然后传给大模型
